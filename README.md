# 📰 Fake News Detection Project

## 📌 Description

This project is a simple Machine Learning application that detects whether a news article is **FAKE** or **REAL**.

It uses basic **Natural Language Processing (NLP)** techniques to analyze the text and make predictions.

---

## ⚙️ How the Project Works

1. The dataset containing news text and labels is loaded
2. The text data is converted into numbers using **TF-IDF Vectorization**
3. The dataset is split into training and testing data
4. A **Logistic Regression** model is trained
5. The model predicts whether the input news is FAKE or REAL

---

## 📂 Files in the Project

* `main.py` → Main Python file that runs the project
* `news.csv` → Dataset file
* `requirements.txt` → List of required libraries

---

## 🛠️ Requirements

Make sure Python is installed on your system.

Install the required libraries using:

```
pip install -r requirements.txt
```

---

## ▶️ How to Run the Project

Step 1: Open terminal in project folder

Step 2: Run the following command:

```
python main.py
```

---

## 🧪 Usage

* After running the program, enter any news text
* The model will predict whether it is **FAKE** or **REAL**

Example:

```
Enter news text: AI will replace humans tomorrow
Prediction: FAKE
```
